

# Lexicon

VADER's dictionary is manually annotated, and each word has a valence score. During the evaluation, a set of rules dynamically adjust this score.

This Lexicon model does not require training; it is directly evaluated.

This used **VADER Sentiment** to be as the lexicon. Essentially, it's a **lexicon** and **rule system** with built-in sentiment **scores (-4 to +4)**. It **doesn't require learning data**, so there's **no need for training**; it directly uses inference and evaluation.

### the test data to evaluate

The test data has been generated by the other teammates:

Our self-built crawler dataset: `./val/source_0/sft_val.json`

Existing open source datasets: `./val/source_1/sft_val.json`

For the labels of the test, this code has used regular expressions to match `\\boxed{positive}`, `\\boxed{negative}` or `\\boxed{neutral}`. We define `positive = 1 neutral=0 negative= -1`, it is a three-class NLP task.

### Problems of the test data

#### Long-tail distribution

We checked the number of each class in the data, and we found they are not balanced.

#### problems of the model

VADER is a binary classification, and its design intent is **positive vs. negative**, not **positive / neutral / negative**.

Neutral is close to 0, but in many sentences:
Slightly positive → is judged as positive
Slightly negative → is judged as negative

Hence it is not good in Three-category tasks, and it cannot recognize the neutral sentences well.

### data analysis & conclusions

It is very similar between the two different datasets, including accuracy and ROC diagrams. Which means the model problem is "systematic," not a matter of chance in the dataset.

The data shows:

```
Very strong for positive
Almost ineffective for neutral
Moderately effective for negative

Positive (performs very well)
Precision ≈ 0.77
Recall ≈ 0.88
F1 ≈ 0.82

Negative (Moderate)
F1 ≈ 0.57 / 0.61 Can identify, but unstable. From the confusion matrix: [-1] → Many are predicted as positive

Neutral (Catastrophic)
F1 ≈ 0.04 / 0.06
Recall ≈ 0.02 ~ 0.04 Basically cannot identify neutral

AUC ≈ 0.6635
Note: Has some discriminative ability but not strong.
```

And we can see **class 0: neutral** (the oringle lines) is very close to the diagonals, which means it judges neutral class randomly. It nearly has no ability to distinguish neutral.

Hence its distinguishment ability is `positive ≫ negative ≫ neutral`, it is easy to judge a sentence is positive.
